NAME
      srtm2scidb - a command line application for converting SRTM HGT data into SciDB multidimensional arrays.

SYNOPSIS

      srtm2scidb <SRTM-hgt-source-file> <output-file>

DESCRIPTION

srtm2scidb is a command line application for converting
SRTM HGT data into SciDB multidimensional arrays.

FURTHER DOCUMENTATION

      srtm2scidb S34W054.hgt name.scidb


For more infomation on SRTM to SciDB, please, visit its main web page at: https://github.com/e-sensing/scietl.
